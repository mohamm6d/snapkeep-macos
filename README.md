# SnapKeep

A modern macOS screenshot manager focused on privacy, simplicity, and on-device processing.

SnapKeep helps you capture, organize, and manage screenshots efficiently without relying on external services or cloud-based processing by default.

---

## Overview

SnapKeep is designed for users who want a clean and fast workflow for handling screenshots on macOS. It combines capture tools, lightweight editing, and file organization into a single menu bar experience.

The app emphasizes:

- Minimal friction
- Local-first architecture
- Clean macOS-native design
- Privacy-focused workflows

---

## Core Features

### Screenshot Capture

- Full-screen and area-based capture
- Multi-display support
- Customizable keyboard shortcuts
- Native macOS-style capture experience

### Annotation & Editing

- Basic drawing and markup tools
- Highlighting and redaction options
- Text overlays and simple shapes
- Crop and adjust screenshots quickly

### Text Recognition (OCR)

- Extract text directly from screenshots
- Copy or export recognized text
- Works fully on-device

### Smart Organization

- Folder-based file management
- Sorting and search capabilities
- Grid and list browsing modes
- Rename, delete, and import support

### Privacy & Security

- Optional file encryption
- Metadata removal for sensitive content
- Local authentication support
- No tracking or analytics

### Sync (Optional)

- Integration with system file syncing (optional)
- Automatic background updates across devices

- **Privacy-first design**

No accounts, no forced cloud usage, no unnecessary complexity.
